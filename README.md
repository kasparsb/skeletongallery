Skeleton gallery
===============

Markup free image/video gallery.

You only need container element for gallery to reside in.

jQuery is dependency

Basic usage

```
gallery = new webit.Viewer({
	container: $('.container'),
	sizeMethod: 'fit',
	images: [
	  '/path/to/image1.jpg',
	  '/path/to/image2.jpg',
	  '/path/to/image3.jpg',
	  ...
	]
);
```
